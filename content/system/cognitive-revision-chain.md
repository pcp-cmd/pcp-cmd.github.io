# Cognitive Revision Chain / 认知修订链

认知修订链把学习过程拆成十个可共事节点：

1. Raw Experience / 原始经验
2. Attention Gate / 注意力筛选
3. Prediction Error / 认知误差
4. Personal Delta / 个人增量
5. Encoding / 结构编码
6. Connection / 知识连接
7. Retrieval / 提取复现
8. Compression / 工具压缩
9. Skill / 流程固化
10. Revision Loop / 反馈修订

这条链条的目的不是解释大脑，而是把学习变成可以被 AI coworker 参与维护的长期系统。
