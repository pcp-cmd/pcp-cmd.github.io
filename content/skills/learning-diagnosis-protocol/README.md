# 可迭代学习诊断技能

这组文件保存用户上传的学习诊断技能。它是本站的理念核心：把学习、写作、设计和 AI 共事中的卡点，转化成诊断、训练输出和可复用资产。

## 放置规则

- `SKILL.md`：中文化后的技能正文，供网站作为核心资产读取。
- `openai.yaml`：保留给本地工具链识别。
- 本说明：用于解释这项技能在网站中的位置。

## 网站用途

1. **Protocol**：作为可迭代修订协议的学习诊断核心。
2. **Math Lab**：把定义卡点、证明跳步和反例缺失转化成卡片。
3. **Manuscripts**：把笔记、复盘和日志组织成可继续修订的手稿。
4. **Works**：把作品反馈、失败版本和修改方向压缩成可复用经验。
