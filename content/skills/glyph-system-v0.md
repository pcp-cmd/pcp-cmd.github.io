# 视觉符号卡片生产 Skill

这个 skill 用于把文章、数学笔记、设计项目转换成统一的手稿式视觉符号。

## Input

- 内容标题
- 核心 thesis
- 所属房间
- 情绪关键词
- 需要表达的动作关系

## Output

- Mother glyph
- Accent color
- One-line visual rule
- Prompt draft
- Revision checklist

## Visual Rules

- 温暖纸张
- 粗黑手绘线
- 低饱和色块
- 不闭合结构
- 留白优先
