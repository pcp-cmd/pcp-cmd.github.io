# 集合论学习索引

这一页是数学分析第一章的入口。它不只是文件目录，而是把集合论的定义、例子、反例、证明拆解和复习卡片组织成一个可重复进入的学习包。

## 当前任务

- 定义：把集合、子集、关系、映射、偏序、等势这些概念写成可闭卷复述的版本。
- 例子与反例：每个定义至少配一个正例和一个反例。
- 证明拆解：把常见证明拆成“要证什么、任取什么、使用哪个定义、如何收尾”。
- 常见误解：记录符号层级混乱、像与逆像混用、偏序与全序混淆等问题。
- 工具卡片：把反复出现的动作压缩成复习卡片。
- 修订记录：把每次卡住的地方，送回下一轮学习。

## 资产流向

```text
原始笔记 -> 结构化讲解 -> 网页可读公式 -> 网站展示 -> 工具卡片 -> 修订记录
```

## 资产类型

- 定义卡片
- 例子 / 反例
- 证明拆解
- 认知误差记录
- 工具箱卡片
- 修订日志
