# 1.1 集合论章节包索引

## 使用顺序

1. `01-set-theory-overview.md`  
   先读总览，知道本章要训练什么。

2. `01-1-set-axioms-learning.md`  
   学集合公理、映射、等价关系。

3. `01-2-order-structures-learning.md`  
   学偏序、全序、上下界、确界原理。

4. `01-3-cardinality-learning.md`  
   学等势、势比较、Cantor 定理。

5. `01-set-theory-solutions.md`  
   做完题再看答案。不要先读答案。

6. `01-set-theory-cards-and-test.md`  
   做闭卷测试，并把卡片加入复习队列。

## 今日 Minimum Action

今天只做三件事：

1. 闭卷写出 `in`、`subset`、映射、等价关系、偏序、等势的定义。
2. 完成闭卷测试第 `1-3` 题。
3. 把错题写成一张 Mistake Card。

## Training Output

你今天要留下一个真实输出：

```text
一页手写或电子版证明：
1. De Morgan 公式
2. 逆像保交
3. 像不保交的反例
```

## Asset To Keep

保留以下资产：

- Definition Card：子集、映射、等价关系、偏序、上确界、等势。
- Example Card：`N` 与 `2N` 等势。
- Counterexample Card：像不保交、偏序不全序、上确界不是最大元。
- Proof Card：Cantor 对角线法。

## Feedback Check

完成后检查：

1. 是否能不用看材料写出 `D={x in X:x notin g(x)}`？
2. 是否能解释为什么 `emptyset subset X`？
3. 是否能区分 `{x}`、`x`、`{x,y}`、`(x,y)`？
4. 是否能说出为什么 `f(A intersection B)` 不一定等于 `f(A) intersection f(B)`？

若任一问题答不上来，回到对应学习文件重读并重做测试。

## 下一章衔接

集合论完成后，进入 `1.2 自然数、整数与有理数`。其中自然数构造会立即使用：

- 集合公理。
- 映射。
- 归纳法。
- 序结构。
- 等价关系。

所以集合论不是背景知识，而是后续数系构造的语言地基。
