# 1.1.1 集合公理讲解

## 1. 外延公理

外延公理说：

```text
若 X 与 Y 有完全相同的元素，则 X=Y。
```

形象例子：两个袋子只看里面的卡片是否相同，不看袋子的名字。`{1,2}` 与 `{2,1}` 是同一个集合。

训练输出：证明集合相等时，写：

```text
任取 u。
u in X <=> ...
          <=> u in Y。
```

## 2. 子集

定义：

```text
X subset Y <=> forall u (u in X => u in Y).
```

形象例子：`X` 是小袋子，`Y` 是大袋子；小袋子里的每张卡片都能在大袋子中找到。

边界例子：

```text
emptyset subset X
```

因为空集没有元素，没有元素能违反条件。

## 3. 配对公理与有序对

配对公理保证 `{x,y}` 是集合。集合无序，所以：

```text
{x,y}={y,x}.
```

为了表达顺序，定义 Kuratowski 有序对：

```text
(x,y)={{x},{x,y}}.
```

它满足：

```text
(x,y)=(a,b) <=> x=a 且 y=b。
```

形象例子：`{北京,上海}` 只是两个城市；`(北京,上海)` 表示从北京到上海。

## 4. 分离公理

分离公理允许从已有集合中筛选：

```text
{u in X : phi(u)}
```

关键：必须先有大集合 `X`。不能随便说“所有满足某性质的对象组成集合”。

形象例子：

```text
{n in N : n 是偶数}
```

合法，因为从 `N` 中筛选。

Russell 悖论的危险写法：

```text
{x : x notin x}
```

没有先给定大集合，不是分离公理允许的构造。

## 5. 交、差、空集

由分离公理：

```text
X intersection Y = {u in X : u in Y}
X \ Y = {u in X : u notin Y}
emptyset = X \ X
```

## 6. 并集公理

并集公理允许把一族集合合并：

```text
union mathcal X = {u : exists A in mathcal X, u in A}.
```

若 `mathcal X={A,B}`，则 `union mathcal X=A union B`。

## 7. 幂集公理

幂集：

```text
P(X)={u : u subset X}.
```

例子：

```text
P({1,2})={emptyset,{1},{2},{1,2}}.
```

## 8. 直积、关系、映射

直积：

```text
X x Y={(x,y): x in X, y in Y}.
```

关系：

```text
R subset X x Y.
```

映射是特殊关系：每个 `x in X` 对应唯一 `y in Y`。

不是映射的两个典型失败：

- 某个输入没有输出。
- 某个输入有两个不同输出。

## 9. 单射、满射、双射

单射：不同输入不会撞到同一个输出。

满射：目标集合每个元素都被命中。

双射：一一对应。

形象例子：

```text
单射：每个人身份证号不同。
满射：每个座位都有人坐。
双射：一人一座，刚好坐满。
```

## 10. 像与逆像

像：

```text
f(A)={f(x): x in A}.
```

逆像：

```text
f^{-1}(B)={x in X : f(x) in B}.
```

核心规律：

- 像保并，但不一定保交。
- 逆像保并、保交、保差。

这就是为什么分析中定义连续性、可测性时经常使用逆像。

## 11. 选择公理

选择公理：给定一族非空集合，可以从每个集合中选出一个元素。

形象例子：

- 无限多双鞋：可按“选左脚”来选，不需要抽象选择。
- 无限多双袜子：没有天然规则，就需要选择公理保证选择函数存在。

## 12. 等价关系

等价关系满足：

```text
自反：x~x
对称：x~y => y~x
传递：x~y 且 y~z => x~z
```

等价类：

```text
[x]={y in X : y~x}.
```

等价关系就是分类规则。商集 `X/~` 是所有分类块组成的集合。
