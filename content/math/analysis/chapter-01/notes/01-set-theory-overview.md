# 1.1 集合论学习总览

对应讲义：`1.1 集合论`

## Current Diagnosis

当前最薄弱环节不是时间，也不是天赋，而是 `schema structure` 和 `tool system`：集合论里定义很多，若只顺着讲义读，很容易把 `in`、`subset`、关系、映射、等价类、上确界、势这些概念堆在一起。我们要把它拆成可训练的模块。

## Training Output

学完本章后，你必须能闭卷完成：

1. 用双包含证明集合相等。
2. 判断一个关系是否是映射、等价关系、偏序、全序。
3. 写出像与逆像对并交差的关系。
4. 区分最大元、上界、上确界。
5. 用双射、单射、满射比较集合的势。
6. 复述 Cantor-Schroder-Bernstein 定理和 Cantor 无最大势定理。

## Asset To Keep

本章要保留四类资产：

- Definition Card：外延公理、分离公理、映射、等价关系、偏序、上确界、等势。
- Example Card：`N` 与偶数集等势、幂集偏序、模 `n` 同余。
- Counterexample Card：真子集不一定更小、像不保交、偏序不一定全序。
- Proof Card：双包含法、De Morgan、Cantor 对角线法。

## 本章结构

### 1.1.1 集合公理

核心问题：哪些集合构造是合法的？

必须掌握：

- 外延公理：集合由元素决定。
- 配对公理：可以构造 `{x,y}`。
- 分离公理：只能从已有集合中筛选。
- 并集公理：一族集合可以取并。
- 幂集公理：所有子集组成集合。
- 直积：`X x Y`。
- 关系：`R subset X x Y`。
- 映射：每个输入有且只有一个输出。
- 等价关系：自反、对称、传递。
- 商集：按等价类分类。

### 1.1.2 集合上的序结构

核心问题：如何抽象地比较大小？

必须掌握：

- 偏序：自反、反对称、传递。
- 全序：任意两个元素可比。
- 上界、下界。
- 最大元、最小元。
- 上确界、下确界。
- 确界原理。
- 单调映射。

### 1.1.3 集合的势

核心问题：无限集合如何比较大小？

必须掌握：

- 等势：存在双射。
- `card X <= card Y`：存在单射 `X -> Y`。
- 单射和满射的等价刻画。
- Cantor-Schroder-Bernstein 定理。
- Cantor 无最大势定理。

## 学习顺序

1. 先学 `in` 与 `subset`，否则后面所有证明都会乱。
2. 再学集合运算和双包含法。
3. 再学关系与映射，尤其像和逆像。
4. 再学等价关系和商集。
5. 再学序结构。
6. 最后学集合的势。

## 本章最重要的 10 个句子

1. 集合由元素决定。
2. 证明集合相等通常证明双包含。
3. `in` 连接元素和集合，`subset` 连接集合和集合。
4. 分离公理只能从已有集合中筛选元素。
5. 映射是特殊关系：每个输入有唯一输出。
6. 逆像比像更稳定。
7. 等价关系就是分类规则。
8. 最大元必须属于集合，上确界不一定属于集合。
9. 无限集合可以和真子集等势。
10. 任意集合的幂集都比它本身更大。
