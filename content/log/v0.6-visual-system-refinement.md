# v0.6 Visual System Refinement

## Change

Rebuilt the homepage visual language around a warmer manuscript surface, four independent room glyphs, and a clearer GSAP motion sequence.

## Decisions

- Keep the English typography close to the Claude-style serif/sans pairing, while using a Chinese serif stack for non-title manuscript text.
- Replace large random card fills with one quiet warm-paper card system and a small clay annotation mark on hover.
- Move the four room illustrations from one-off path strings to `glyphType`, then render each room through a reusable SVG glyph function.
- Add a hero signal sweep so the opening animation reads as write, connect, activate.
- Exclude the Cognitive Revision Chain cards from the generic reveal batch so their phase rhythm stays intentional.

## Reason

The v0.5 page had the right theme but still felt visually uneven: the hero animation was thin, room SVGs were too similar, and several card colors competed with the manuscript atmosphere.
