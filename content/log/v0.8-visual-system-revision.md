# v0.8 Visual System Revision

## Change

Reworked Aleksi Lab from a concept-heavy homepage into a personal research site with a clearer content flow, reusable illustration slots, and a stricter warm editorial visual system.

## Decisions

- Move Plog directly after the hero so the site reads as an active research stream.
- Replace the complex inline SVG hero with a static WebP illustration slot.
- Replace the four room SVG glyphs with room illustration cards.
- Use Newsreader, Instrument Sans, and IBM Plex Mono for the main typography system.
- Establish a Paper / Graphite / Oat / Cactus / Olive / Clay / Slate palette with tone classes.
- Keep GSAP focused on entrance rhythm, scroll reveal, hover response, and subtle image movement.

## Reason

The previous version had the right subject matter, but it was still too much like a concept page. v0.8 makes the site easier to extend as a long-term research archive: content first, system second, visual assets reusable.
