# v1.2.5 — Card Material Unification

This revision addresses the "cards feel out of place" issue in the dark Claude direction.

## Changes

- Removed large-area colored card bodies.
- Unified all cards as warm graphite / dark paper surfaces.
- Moved Claude palette usage to top hairlines, annotation dots, metadata accents, and active states.
- Removed glow-like box shadows from card dots/underlines.
- Increased hierarchy contrast with ivory titles and warm gray body copy.
- Kept clay as the primary action/direction color.
- Kept ochre, sage, blue, and violet as restrained indexing tones.

## Principle

Cards should feel like the same dark manuscript material with different colored index tabs, not a set of colorful UI blocks.
