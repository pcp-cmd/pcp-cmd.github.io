# v1.7.0 Works Review Fix

本次修复针对用户截图中的两个主要问题：

1. `work-detail.html` 的“作品评分”标题与分数卡片视觉距离过近，标题字号继承错误，导致像是压到下方评分。
2. `works.html` 展览墙的选中卡片存在文字安全区不足、长来源文字挤压、卡片绿色 / 蓝色偏色导致质感发脏的问题。

## 已改动

- 新增 `assets/css/100-works-review-fix.css`，并在 `styles.css` 最后引入，确保它覆盖旧的 v1.6.x 累积样式。
- 将 `.work-scores` 改成完整面板，并重写评分标题、评分网格、评分卡片的字体与间距。
- 将 Works 展览墙卡片统一到 warm charcoal 色系，去除绿蓝偏色。
- 放大选中卡片高度与文字安全区，给标题、摘要、来源和按钮分别留出固定层级空间。
- Works 卡片摘要使用短句裁切，详情页保留完整文字，避免展览卡里文字被挤压。
- 长来源/媒介信息使用单行省略，避免挤占“进入作品”按钮。
- 选中卡片与 hover 加入克制的 clay 边框反馈，不做刺眼光边。

## 自查重点

- 详情页：评分标题不再压到数字；6 个评分卡片在桌面端保持一行，窄屏自动换行。
- Works 页：选中卡片内部文字不被图片或底部按钮遮挡；底部未选中卡片弱化，不再抢视觉。
- 色彩：整体保持深色档案墙，不再出现突兀绿色、蓝灰色卡片。
