# v1.2.3 Dark Claude Discipline Pass

This revision responds to the dark Claude screenshots and the “light pollution” feedback.

## What changed

1. Rebuilt dark material tokens around warm black, graphite cards, ivory type, and clay action accents.
2. Removed glow-like behavior: no text shadows, no color-wash card blocks, no strong hover lift.
3. Cards now share one warm graphite material. Claude colors only appear as top hairlines, small marks, borders, and metadata accents.
4. GSAP motion was slowed and reduced: reveal starts at 0.72+ opacity, y movement is 8–10px, line drawing is slower, hover lift is only -1px.
5. Chain stage headers were forced into a vertical editorial layout to prevent title/summary collisions.
6. Menu overlay remains disabled; direct header navigation remains.

## Motion constitution

- No glow, pulse, bounce, scanline, or looping flourish.
- Reveal is a slow manuscript fade, not a flash.
- Hover is a light touch, not a product-card performance.
- SVG lines draw like ink, not neon.
