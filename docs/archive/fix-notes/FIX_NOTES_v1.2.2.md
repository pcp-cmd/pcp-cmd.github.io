# Aleksi Lab v1.2.2 — Dark Claude Contrast + Accent Card Pass

This revision responds to the dark-theme feedback:

1. Typography now uses ivory for primary meaning and Claude terracotta/clay for orientation, links, section labels, and metadata.
2. Cards are no longer only graphite. They use the Claude-like secondary palette through restrained material surfaces: ochre, sage, dusty blue, violet, and clay.
3. Accent color is visible as top rules, borders, metadata, dots, and very subtle card temperature, not as loud SaaS blocks.
4. Hero and article surfaces use stronger clay + ivory contrast.
5. Section shells carry subtle radial Claude palette temperature in dark mode.
